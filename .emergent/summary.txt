**what currently exists?**
The application is a full-stack React/FastAPI app with a MongoDB database. The school management system is highly functional.
- **School Dashboard:** Features a Gameday section showing live and scheduled games for the current day, a full calendar, and a tabbed interface for Schedule, Stats, Members, and Edit School.
- **Season Management:** School admins can create seasons with specific sports, genders (Men/Women), and levels (Varsity/Sub-Varsity). They can manage schedules, opponents, and rosters. The opponent creation process now includes a searchable dropdown to find existing opponents or import other schools on the StatMoose platform.
- **Admin Dashboard:** The main admin can view platform-wide statistics (total users, schools, teams, games). It features collapsible sections for viewing a detailed list of all participating schools (including their classification, members, and seasons) and all users.
- **School Creation:** New schools can be created with a unique, non-editable 3-10 character ID. The signup form includes a mandatory logo upload and a classification dropdown (High School, College, etc.).
- **Game Status:** The Gameday UI shows if a game is LIVE NOW (with live score and period) or scheduled. A warning dialog prevents multiple users from opening the stat tracker for the same in-progress game.
