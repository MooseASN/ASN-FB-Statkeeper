            6.  Select the tackler, or no tackle.

        *   Create a unified input area for plays.

        *   **Run Plays:** After collecting rusher and gain/loss data, allow the user to select the tackler.

        *   First downs should be calculated automatically based on the result of the play; no manual 1st down button is needed.
